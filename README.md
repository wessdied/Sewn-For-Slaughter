# Sewn-For-Slaughter
An in-progress multiplayer horror game developed in Unity.
All Rights Reserved

## Features
- Core movement and collision using C#
- Custom stamina and health UI
- Modular 3D assets and animations from Blender
- Multiplayer structure in early development

## Tools Used
- Unity (2025)
- C#
- Blender (for models/rigs/animations)
- Git for version control

## How to Run
1. Clone the repository
2. Open the folder in Unity Hub
3. Open the main scene and press Play
